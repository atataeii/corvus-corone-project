Dataset: Corvus corone

The raw datasets are not uploaded to GitHub due to file size limitations.

Each group member should place the files manually inside `data/raw/` using these exact filenames:

- `data/raw/corvus_corone.csv`
- `data/raw/knmi_de_bilt_daily_weather.csv`

The notebook expects these paths when it loads the observation data and KNMI daily weather data. If the notebook does not open correctly on GitHub, download the repository and run the notebook locally on your own system.
