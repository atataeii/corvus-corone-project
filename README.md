# Corvus corone Biodiversity Prediction Project

## Team Members
- Matin
- Mohamad
- Dima

## Species
Corvus corone

## Goal
Predict whether Corvus corone is observed at a specific location and date.

## Data Sources
- GBIF species observations
- CBS habitat/ecosystem data
- KNMI weather data

## Method
This project uses a classification approach:

- 0 = no observation
- 1 = observation present

## Project Structure
- data/raw: original data
- data/processed: cleaned and feature data
- notebooks: main analysis notebook
- scripts: Python scripts
- results: figures and tables
- poster: final factsheet/poster
